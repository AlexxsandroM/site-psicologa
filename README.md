# Site Psicóloga - Dra. Maria Oliveira

Website profissional para consultório de psicologia clínica.

## Tecnologias

- React + TypeScript
- Vite
- Lucide React (ícones)
- CSS puro (sem frameworks)

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

```bash
npm run preview
```
