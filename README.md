# Site da Psicologa Waldirene Dias

Projeto pessoal criado para uma profissional de psicologia clinica com dois objetivos principais: apresentar de forma clara a abordagem de trabalho e direcionar visitantes para contato rapido via WhatsApp.

## Visao Geral
- Landing page com hero, seccao Sobre, Servicos e pagina dedicada a abordagem (Gestalt-terapia).
- Botao de contato rapido para WhatsApp em varios pontos da pagina.
- Menu fixo com logo, navegacao por ancora e logo clicavel para voltar ao inicio.
- Paginacao simples via React Router (home e abordagem).

## Principais Recursos
- Hero com chamada para acao e links para agendamento/abordagem.
- Destaque para consultorio presencial e atendimentos online.
- Seccao de servicos (Terapia Individual, Presencial, Consultas Online) com descricoes curtas.
- Rodape com redes sociais, telefone, e-mail e endereco.

## Tecnologias
- React + TypeScript
- Vite
- React Router
- Lucide React (icones)
- CSS puro

## Scripts
```bash
npm install      # instala dependencias
npm run dev      # desenvolvimento
npm run build    # build de producao
npm run preview  # servidor local do build
```

## Estrutura
- src/pages/Home.tsx: landing page principal
- src/pages/Abordagem.tsx: pagina detalhada da abordagem Gestalt
- src/App.tsx: rotas e layout com header fixo
- src/App.css: estilos globais
- src/assets/: imagens e logo
